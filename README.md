# Tienda Online AWS

Proyecto de una tienda virtual de ropa desarrollado con Angular y AWS como parte del proceso de práctica.

## Descripción

Esta aplicación permite visualizar productos de ropa, agregarlos al carrito de compras y calcular el valor total de la compra.

El proyecto será desarrollado utilizando:

* Angular (Frontend)
* AWS Lambda
* API Gateway
* DynamoDB
* GitHub

## Funcionalidades implementadas

* Visualización de productos
* Agregar productos al carrito
* Eliminar productos del carrito
* Cálculo del total de compra
* Interfaz gráfica básica

## Tecnologías utilizadas

* Angular
* TypeScript
* HTML
* CSS
* Git
* GitHub
